# The-Streaming-Service

The Streaming Service is a project that allows the user to choose what media they want to play - music or film. 

An interface and 5 classes are required to be built in this project - DigitalContent, Film, Music, StreamingService, and Client. Required methods and instance variables are listed within the assignment, and a menu with 5 options to choose from is required to be built for the user to select their preference using keywords that matches the media. 

No actual media is played, but the correct media name is shown that is being "streamed". 

This assignment scored 100/100.
